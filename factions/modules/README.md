# Modules

This folder contains the campaign’s **narrative rails** and major optional modules.

## Core Rails
- **/modules/acts/** — the 5‑Act Civil War backbone:
  - ACT_01_BATTLE_OF_WHITERUN
  - ACT_02_THE_HOLDS_SHIFT
  - ACT_03_THE_CITY_CRISIS_WAVE
  - ACT_04_THE_FINAL_STORM
  - ACT_05_THE_TRUE_ENEMY
  - EPILOGUE

Each Act file should include:
- premise + tone
- scene menu
- branch notes
- which clocks tick and why
- where Side Plot C can plug in

## Optional Modules
- **/modules/daedric_princes/** — Daedric questlines, artifacts, champion aspects, influence clocks
- **/modules/dragonbreaks_creatures_companions/** — Dragonbreak moments, creature blocks, companion framework
- **/modules/locations/** — hold/city packets
